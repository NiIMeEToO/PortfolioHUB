# PortfolioHUB

Este repositório tem como objetivo reunir meus projetos acadêmicos e pessoais, utilizando boas práticas de versionamento com Git e GitHub.

## Objetivo

Organizar projetos, documentos, slides e páginas web em um único ambiente profissional, facilitando o compartilhamento com professores, colegas e recrutadores.

## Estrutura do Repositório

- `projetos-academicos/`: projetos desenvolvidos durante o curso.
- `projetos-pessoais/`: projetos criados para estudo e prática.
- `documentos/`: arquivos de apoio e documentação.
- `slides/`: apresentações relacionadas aos projetos.
- `index.html`: página inicial publicada no GitHub Pages.

## Tecnologias Utilizadas

- Git
- GitHub
- GitHub Pages
- HTML
- CSS
